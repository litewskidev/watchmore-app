## watchmore app ##
This product uses the TMDB API but is not endorsed or certified by TMDB.

Multi-page 'streaming' app with login, sign up, watch list, movies, TV shows, collections, hubs and search.

# Build with:
- HTML5
- JavaScript
- React
- React Router
- Redux
- Express
- Firebase
- Slick
- Sass
- CSS Flexbox
